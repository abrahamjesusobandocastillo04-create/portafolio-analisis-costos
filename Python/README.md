# Python – Soporte a Proyectos de Logística, Mantenimiento y Costos

## Descripción
Scripts en Python utilizados para automatizar y controlar información de proyectos en un entorno agroindustrial,
incluyendo logística, mantenimiento y seguimiento de centros de costos.

## Funcionalidades
- Limpieza, validación y análisis de datos operativos
- Control y reporte de mantenimiento
- Identificación y validación de órdenes de compra y servicios por centros de mejora
- Automatización del seguimiento de costos por proyecto
- Preparación de datos para análisis y visualización en Power BI

## Herramientas
- Python
- pandas
- numpy
- Jupyter Notebook
