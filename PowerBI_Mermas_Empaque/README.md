# Power BI – Control de Mermas de Empaque

Dashboard desarrollado en Power BI para el control y análisis de mermas de 
empaque durante la campaña de mango 2025–2026 en el sector agroindustrial.

## Objetivo del Dashboard
Monitorear el nivel de merma por producto de empaque y embalaje para evaluar su impacto 
económico frente a los límites establecidos, permitiendo identificar 
excedentes y oportunidades de mejora.

## Principales indicadores
- Merma por producto de empaque y embalaje
- Tipos de merma: producción, calidad y proveedor
- Excedente de merma en valor monetario
- Ratio de consumo vs producción
- Análisis por semana y rango de fechas

## Herramientas utilizadas
- Power BI
- Power Query
- DAX
- Modelado de datos
